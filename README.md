# 36krfe.github.io
36氪前端官方博客

## 原创博文更新指南

1. fork 到自己的仓库。
2. 在 posts 目录下新建文章，如``2017-7-21-36kr-test.md``，可以参考其他文章。
3. 编辑完提交 PR , 审核后发布

```
---
layout: post
title: 标题
categories: [分类]
tags: [标签]
fullview: true
---

# 正文
```

## 英文博文翻译指南

1. fork 到自己的仓库。
2. 在 originals 目录下新建英文原文，如``2017-7-21-36kr-english-test.md``，可以参考其他文章。
3. 翻译完提交 PR , 校对后发布。

```
---
layout: original
title: 标题
categories: [分类]
tags: [标签]
fullview: true
---

# 正文
```
